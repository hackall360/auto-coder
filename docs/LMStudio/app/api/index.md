You can serve local LLMs from LM Studio's Developer tab, either on localhost or on the network.

LM Studio's APIs can be used through an [OpenAI compatibility mode](/docs/LMStudio/app/api/endpoints/openai), enhanced [REST API](/docs/LMStudio/app/api/endpoints/rest), or through a client library like [lmstudio-js](/docs/LMStudio/developer/typescript).

#### API options

- [TypeScript SDK](/docs/LMStudio/developer/typescript) - `lmstudio-js`
- [Python SDK](/docs/LMStudio/developer/python) - `lmstudio-python`
- [LM Studio REST API](/docs/LMStudio/app/api/endpoints/rest) (new, in beta)
- [OpenAI Compatibility endpoints](/docs/LMStudio/app/api/endpoints/openai)
